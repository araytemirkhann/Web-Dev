//Чтобы создать переменную в JavaScript, используйте letключевое слово. let message; //Теперь мы можем поместить в него некоторые данные с помощью оператора присваивания =: let message; message = 'Hello'; let message; message = 'Hello!'; alert(message); // shows the variable content let message = 'Hello!'; // define the variable and assign the value alert(message); // Hello! //Мы также можем объявить несколько переменных в одной строке: let user = 'John', age = 25, message = 'Hello'; //Многострочный вариант let user = 'John'; let age = 25; let message = 'Hello'; let user = 'John', age = 25, message = 'Hello'; //в стиле «сначала запятая»: let user = 'John' , age = 25 , message = 'Hello'; //varвместоlet var message = 'Hello'; //Ключевое varслово почти такое же, как let. //Он также объявляет переменную, но немного другим, «старомодным» способом. //При изменении значения старые данные удаляются из переменной: let message; message = 'Hello!'; message = 'World!'; // value changed alert(message); Объявление дважды вызывает ошибку let message = "This"; let message = "That"; let hello = 'Hello world!'; let message; message = hello; alert(hello); // Hello world! alert(message); // Hello world! //Чтобы объявить постоянную (неизменную) переменную, используйте constвместо let: const myBirthday = '18.04.1982'; //Переменные, объявленные с помощью const, называются «константами». Их нельзя переназначить. Попытка сделать это вызовет ошибку: const myBirthday = '18.04.1982'; myBirthday = '01.01.2001'; // error, can't reassign the constant! const COLOR_RED = "#F00"; const COLOR_GREEN = "#0F0"; const COLOR_BLUE = "#00F"; const COLOR_ORANGE = "#FF7F00"; let color = COLOR_ORANGE; alert(color); // #FF7F00 //let– это современное объявление переменной. //var– объявление переменной старой школы. Обычно мы его вообще не используем, но мы расскажем о тонких отличиях из letглавы Старая "var" на тот случай, если они вам понадобятся. //const– похоже на let, но значение переменной изменить нельзя.
