alert( Number(" 123 ") ); // 123 alert( Number("123z") ); // NaN (error reading a number at "z") alert( Number(true) ); // 1 alert( Number(false) ); // 0 alert( Boolean(1) ); // true alert( Boolean(0) ); // false alert( Boolean("hello") ); // true alert( Boolean("") ); // false String Conversion– Происходит, когда мы что-то выводим. Может выполняться с String(value). Numeric Conversion- Встречается в математических операциях. Может выполняться с Number(value). Пустая строка становится 0. Выдает ошибку NaN. Boolean Conversion– Встречается в логических операциях. Может выполняться с Boolean(value).
